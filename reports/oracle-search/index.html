Exp 7 — Oracle ablation ceiling: C+/C− pool vs plain attribution

Best-possible ablation (eval-set greedy search) — does the C+/C− pool have a higher ceiling?

Setup
Task. Binary sentiment classification of movie reviews (IMDB-derived). The base LLM is frozen; a linear CLS head reads the final-token hidden state and predicts positive / negative.
Spurious setups (two regimes). bad (amplify): the ordinary word "bad" is made to correlate with the negative label. zxq (construct): a non-semantic token "zxq" is injected and made perfectly correlated with the negative label in training.
Models. Qwen2.5-0.5B-Instruct and Llama-3.1-8B-Instruct.
Masks. Sparse per-neuron MLP masks (hard-concrete L0) are trained on the frozen base + CLS head to solve the task. All interpretation and ablation here run with these learned masks turned ON — we analyze/remove neurons on top of the mask, not instead of it.
Evaluation. Balanced 2×2 validation set (label × cue-present), 2000 rows. gap = majority-group accuracy − minority-group accuracy, where the minority groups are pos+cue and neg+no-cue. A large gap means the model is leaning on the spurious cue.
This experiment (oracle ablation ceiling). To find the best possible ablation for each attribution method, we search the eval set directly (deliberate oracle/'cheating') via greedy forward selection that maximizes worst-group accuracy. Two pools: unsigned = top-150 by |grad×act| (plain attribution, pre-C+/C−); C+/C− = top-75 C+ ∪ top-75 C−. If the C+/C− pool reaches a higher ceiling, the signed decomposition exposes a strictly better set of removable neurons than plain attribution. (Upper bounds only — not a deployable method.)
Qwen/Qwen2.5-0.5B-Instruct | cue=bad | maximize worst-group accuracy (greedy forward, eval-set oracle)
Key result (oracle ceilings, full-val worst-group acc). baseline 0.332 → unsigned-top150 0.538 (Δ+0.206, gap +0.414), C+/C−-pool 0.442 (Δ+0.110, gap +0.473). Higher ceiling: unsigned |A|. These are eval-set oracle upper bounds (greedy forward selection), so they bound the best any selection rule on that pool could achieve.
